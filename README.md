# GitHub Profile App Project

## Project 주제

: 본인의 GitHub 프로필을 보여주는 App 만들기

- 화면 상단에 해당 프로필의 이미지, 사용자 이름, 팔로우 수와 팔로잉 수 출력
- 화면 하단에 해당 프로필의 레퍼지토리 목록을 출력
